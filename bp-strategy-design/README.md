# bp-strategy-design

**Strategy Design: General approaches and methodologies**

This plugin provides foundational strategic analysis frameworks for understanding markets, competition, and customers.

## Skills (13)

| Skill | Description |
|-------|-------------|
| **pyramid-principle** | Structured thinking and MECE problem solving (Barbara Minto) |
| **pest-analysis** | Macro environment analysis: Political, Economic, Social, Technological |
| **porters-five-forces** | Industry competitive analysis (Michael Porter) |
| **capability-assessment** | Competitive capability mapping with radar charts |
| **market-definition** | Market boundary definition and segmentation |
| **market-entry-exit** | Entry/exit barrier analysis |
| **value-chain** | Industry value chain analysis |
| **competitor-analysis** | Structured competitor comparison template |
| **root-cause-analysis** | Logic tree approach to problem diagnosis |
| **conversion-waterfall** | Customer conversion funnel analysis |
| **customer-experience** | Customer journey and touchpoint analysis |
| **customer-segmentation** | Customer grouping by needs and behaviors |
| **kpc-comb** | Key Purchase Criteria visualization |

## Commands (5)

| Command | Description |
|---------|-------------|
| `/analyze-market` | Full market analysis: Definition → Competition → Customers |
| `/analyze-competition` | Porter's Five Forces → Competitor Analysis → ROS/RMS |
| `/analyze-customers` | Segmentation → Experience → Conversion Waterfall |
| `/diagnose-problem` | Root Cause Analysis with logic tree |
| `/assess-capabilities` | Capability Assessment with radar charts |

## Quick Selection

| Question | Start With |
|----------|-----------|
| What's happening in the market? | `pest-analysis` → `porters-five-forces` |
| Who are we competing against? | `competitor-analysis` → `capability-assessment` |
| Who are our customers? | `customer-segmentation` → `kpc-comb` |
| Why is this problem happening? | `root-cause-analysis` |

## Example Usage

```
/analyze-market Electric vehicle charging infrastructure in Southeast Asia

/analyze-competition SaaS project management tools

/diagnose-problem Customer churn rate doubled in Q4
```
