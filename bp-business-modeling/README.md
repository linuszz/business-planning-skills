# bp-business-modeling

**Business & Organization Modeling: Strategy articulation**

Tools for articulating strategy, mapping activities, and defining organizational structure.

## Skills (7)

| Skill | Description |
|-------|-------------|
| **strategy-articulation** | Vision → Mission → Capabilities → Initiatives pyramid |
| **swot-analysis** | Strengths, Weaknesses, Opportunities, Threats |
| **activity-maps** | Visual summary of all company activities by value chain |
| **business-definition** | Unified vs separated business entities framework |
| **partnering-maps** | Build vs buy vs partner decision framework |
| **asset-extension** | New business opportunities from existing assets |
| **raci-analysis** | Responsible-Accountable-Consulted-Informed matrix |

## Commands (4)

| Command | Description |
|---------|-------------|
| `/articulate-strategy` | Vision → Mission → SWOT → Initiatives |
| `/map-activities` | Activity Maps → Business Definition |
| `/plan-partnerships` | Partnering Maps → Asset Extension |
| `/clarify-roles` | RACI Analysis for process ownership |

## Quick Selection

| Question | Start With |
|----------|-----------|
| What's our strategic direction? | `strategy-articulation` |
| What are our strengths/weaknesses? | `swot-analysis` |
| What do we actually do? | `activity-maps` |
| Should we build, buy, or partner? | `partnering-maps` |
| What new businesses can we enter? | `asset-extension` |
| Who's responsible for what? | `raci-analysis` |

## Framework: Strategy Articulation Map

```
┌─────────────────────────────────────────────┐
│                  VISION                      │
│     Where do we want to be in 5-10 years?   │
├─────────────────────────────────────────────┤
│                  MISSION                     │
│        Why do we exist? What's our purpose? │
├─────────────────────────────────────────────┤
│              COMPETITIVE ADVANTAGE           │
│      What makes us win vs competitors?      │
├─────────────────────────────────────────────┤
│               CAPABILITIES                   │
│    What must we be great at to deliver?     │
├─────────────────────────────────────────────┤
│               INITIATIVES                    │
│   What specific actions will we take?       │
└─────────────────────────────────────────────┘
```

## Framework: RACI Matrix

| Role | Definition |
|------|------------|
| **R**esponsible | Does the work |
| **A**ccountable | Owns the outcome (single person) |
| **C**onsulted | Provides input before decision |
| **I**nformed | Notified after decision |

## Example Usage

```
/articulate-strategy Fintech startup targeting SMB payments

/map-activities Our e-commerce fulfillment operation

/plan-partnerships We need to add AI capabilities but lack talent

/clarify-roles Our product development process is unclear
```
