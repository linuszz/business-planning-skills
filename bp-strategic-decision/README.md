# bp-strategic-decision

**Strategic Decision Making: Prioritization and risk assessment**

Tools for making strategic choices under uncertainty with limited resources.

## Skills (6)

| Skill | Description |
|-------|-------------|
| **growth-share-matrix** | BCG Matrix: Stars, Cash Cows, Dogs, Question Marks |
| **portfolio-matrix** | NPV vs Ease of Implementation for project prioritization |
| **value-disciplines** | Treacy & Wiersema: Product Leadership, Operational Excellence, Customer Intimacy |
| **prioritization-funnel** | Structured opportunity evaluation and ranking |
| **scenario-development** | Multiple future scenario planning (Shell methodology) |
| **risk-matrix** | Risk identification, prioritization, and mitigation planning |

## Commands (3)

| Command | Description |
|---------|-------------|
| `/prioritize-projects` | Funnel → Portfolio Matrix → Risk Matrix |
| `/assess-portfolio` | Growth Share Matrix → Value Disciplines |
| `/plan-scenarios` | Scenario Development with key uncertainties |

## Quick Selection

| Question | Start With |
|----------|-----------|
| Which projects should we fund? | `prioritization-funnel` → `portfolio-matrix` |
| How healthy is our portfolio? | `growth-share-matrix` |
| What's our competitive strategy? | `value-disciplines` |
| What could go wrong? | `risk-matrix` |
| What might the future look like? | `scenario-development` |

## Framework: Prioritization Funnel

```
┌─────────────────────────────────────┐
│        ALL OPPORTUNITIES            │ ← 20-50 options
├─────────────────────────────────────┤
│   Strategic Fit Filter              │ ← Remove misaligned
├─────────────────────────────────────┤
│   Feasibility Filter                │ ← Remove unachievable
├─────────────────────────────────────┤
│   Financial Impact Filter           │ ← Remove low ROI
├─────────────────────────────────────┤
│   Risk Assessment                   │ ← Flag high-risk
├─────────────────────────────────────┤
│        PRIORITY PROJECTS            │ ← 3-7 priorities
└─────────────────────────────────────┘
```

## Example Usage

```
/prioritize-projects We have 12 growth initiatives and budget for 4

/assess-portfolio Our 8 business units across 3 regions

/plan-scenarios AI regulation impact on our SaaS business
```
